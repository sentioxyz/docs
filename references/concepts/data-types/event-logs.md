# 📕 Event Logs

## Log Levels

Sentio allows users to submit and search for logs. Logs naturally have 5 levels:

* DEBUG
* INFO
* WARNING
* ERROR
* CRITICAL

## System Labels

Sentio also attach system labels automatically to the log, including:

* Contract
* Address
* Chain

## Filter Logs on UI

Using the menu on the left hand side, users can filter the log based on [#log-levels](event-logs.md#log-levels "mention")and [#system-labels](event-logs.md#system-labels "mention")
