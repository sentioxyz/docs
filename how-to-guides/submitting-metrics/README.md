# ➡ Submitting Metrics

Users write [processors](../../developer-guides/sdk-guide/processor-basic.md) using the API within the SDK to submit metrics ([Counter](../../references/concepts/metrics.md#counter) or [Gauge](../../references/concepts/metrics.md#gauge)).  Processors are **triggered** by user-defined conditions (for **both backfill and real time following**). In this section, we give a few examples of HOW-TOs for different chains.&#x20;
