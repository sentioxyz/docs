You can import other projects' data into another project and use the data in Data Studio, Dashboard and other places.

The imported projects could be either Sentio Processor or Subgraph Project Project.

## Import projects

You can manage and import project through **Project -> Data Source -> External Project**.

![](https://files.readme.io/4f2abd2eed333d2d4fd82ce0b1e623c86cbe122db0425fc043afba788027354b-image.png)

And search to import projects

<Image align="center" width="80% " src="https://files.readme.io/f8b7baa80ba518b9e4e97ca11791f283db432765c55ab62973d059cf4b89712a-image.png" />

You can import  multiple project at once, if you import once a time then you can choose to set alias for the project for easier access later.

## Access External Data

Now in SQL Data Studio, you can see table from other projects, you could freely query by joining data from all those projects.
